# Misja na Talerzu — Landing Page Design

## Overview

Single-page landing page for "Misja na Talerzu" — a culinary workshop project in Oswiecim, Poland. Individual cooking workshops (1:1) focused on healthy, practical cooking skills.

## Tech Stack

- Pure HTML + CSS (no frameworks, no build step)
- Single `index.html` with linked `style.css`
- Static hosting ready (GitHub Pages / Netlify)
